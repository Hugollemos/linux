como testar arquivos:
```
- testando se o arquivo é execultavel 
test -x arquivo; echo $?
  '   '      '          '
  '   '      '          - No padrão Unix, um código de saída igual a 0 significa que o último comando foi '      '              executado sem erros. Retorno zero é uma condição especial de sucesso
  '   '- flag para ver se o arquvio é testavel.         
  - parametro para testar algo, também pode ser usado no lugar de "test" o [[ ]]                  
```
